# LoginCustomView
A custom login view

## Setup
#### Add it in your root build.gradle at the end of repositories:
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
#### Add the dependency
```
dependencies {
	        implementation 'com.github.DanielZusev:LoginCustomView:1.00.01'
	}
```
